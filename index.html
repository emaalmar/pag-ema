Introducción a Git y GitHub

¿Qué es Git Bash?

Git es un sistema de control de versiones distribuido que permite registrar cambios realizados en el código. El control de versiones se lleva a cabo mediante "commits", lo que te permite tener un historial de cambios. Además, se pueden usar ramas para trabajar sin afectar la rama principal, lo que te da la ventaja de experimentar sin riesgo. Git se utiliza principalmente para manejar archivos en local.

¿Qué es GitHub?

GitHub es una plataforma en línea que permite almacenar, compartir y colaborar en proyectos de código, ya sea de forma individual o grupal. Se basa en el sistema de control de versiones de Git, que permite llevar un seguimiento detallado del progreso, descargar una copia del código, realizar cambios y subirlos al repositorio compartido. GitHub maneja los archivos en “la nube”.

Introduccion a HTML

¿Que es HTML ?

HTML (HyperText Markup Language o Lenguaje de Marcas de Hipertexto) es el lenguaje estándar para estructurar contenido en la Web. Generalmente, se combina con CSS (para el estilo) y JavaScript (para la funcionalidad).

Anatomía de un elemento HTML

La anatomía de un elemento HTML tiene tres partes:

  1. Etiqueta de apertura: Indica el inicio del elemento (<p> para un párrafo).
  2. Contenido: Texto o elementos incluidos dentro de la etiqueta.
  3. Etiqueta de cierre: Indica el fin del elemento (</p> para un párrafo).

Atributos HTML

Los atributos HTML son palabras especiales que se utilizan para modificar la apariencia o el comportamiento de un elemento en HTML. Se especifican dentro de la etiqueta de apertura de un elemento y constan de un nombre y un valor, separados por un signo igual, ejemplo de atributos:

  • class: Permite asignar un elemento a una o varias clases
  • style: Define el estilo de un elemento HTML, como su color, fuente o tamaño.
  • title: Contiene información adicional sobre el contenido de un elemento.
  • lang: Establece el idioma base de un documento
  • controls: Indica si el navegador debe mostrar controles de reproducción al usuario
  • coords: Especifica las coordenadas del área.
  • data: Especifica la URL del recurso
  • id: Define un identificador único que no debe repetirse en todo el documento

Los atributos tambien pueden tener elementos:

Orden de atributos

Se recomienda este orden para facilitar la lectura del codigo

  • class
  • id, name
  • data-*
  • src,for,type,href,value
  • title,alt
  • role,aria-*
  • tabinex
  • style

Los atributos que son comunmente utilizados para identificar elementos deberian de ir primero : -class,id,name y atributos de datos, los atributos diversos unicos para especificar elementos vienen en segundo lugar, seguidos de atributos de accesibilidad y estilos relativos

¿Qué es CSS?

CSS (Cascading Style Sheets o Hojas de Estilo en Cascada) define cómo se visualizan los elementos de HTML en pantalla. Es esencial para el diseño web, haciéndolo responsivo y fácil de mantener.

Caracteristicas

  • Diseño:
    Definir fuentes,colores,estilos, disposicion de elementos
  • Responsivo:
    Adaptar la pantalla a diferentes dispositivos
  • Independencia:
    Facilidad de modificar estilos sin tocar el HTML
  • Mantenimiento:
    Modificar múltiples páginas a través de un único archivo CSS

Sintaxis básica de CSS

  • Un selector: define a qué elementos se aplicarán las reglas.

  • Un bloque de declaraciones: contiene propiedades y valores.

    Ejemplo:

    selector { propiedad: valor; }

  • Separación clara: las propiedades se separan con `;` y las reglas entre `{ }`.

Uso de CSS

Herencia

Algunos valores de las propiedades CSS de los elementos padre los heredan los elementos descendientes.

Selectores

Un selector CSS es la primera parte de una regla CSS que indica al navegador qué elementos HTML se seleccionan para aplicarles dicha regla. Los más comunes son:

  • Etiqueta: Como <h1>, <body>, <p>, etc.
  • Clase: Indicado con el atributo class="mi-clase" en HTML.

Se puede hacer combinación de selectores mientras cumpla la regla de que cada miembro esté dentro del miembro anterior. Ejemplo:

p span { color: red; }

Especificidad

La especificidad es el modo que tiene el navegador de decidir qué regla se aplica si diversas reglas tienen selectores diferentes pero podrían aplicarse a un mismo elemento.
